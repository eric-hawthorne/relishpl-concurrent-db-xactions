Trial-Program Name

Identify your simple trial relish program by giving it a program name. You can also type in a brief description of your program.

Program names must conform to the following syntax rules: lowercase alphanumeric name with first character a letter, and only underscores '_' permitted between words in the name.

Examples

	mathfun
	program1
	my_simple_trial_program
	test_for_loops

Each differently named program in the trial environment gets its own database, in which its persistent data is stored.

Advanced Feature:

If you ran the relish simple program trial environment with a relish command like:

relish -web 8080 relish.pl2012/playground web

and you typed in a trial program that you named mathfun, then browsing to http://localhost:8080/?artfct=mathfun will start up the trial environment with the mathfun program loaded, and your new run of the program will still have access to whatever persistent data was created by that previous run of the mathfun program.
