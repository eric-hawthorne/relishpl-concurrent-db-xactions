Binary Distributions

Click one of the links below to download a relish binary distribution for your computing platform. Note that the linux binary distributions require separate installation of sqlite3 (version 3.5 or greater) if sqlite3 is not already present on the computer. Apple computers already have sqlite3 installed.

relish_0.0.9.darwin_amd64.tar.gz (MAC OSX)

relish_0.0.9.linux_amd64.tar.gz (64-bit linux - if uname -a reports x86-64)

relish_0.0.9.linux_i386.tar.gz (32-bit linux)

You can install relish under your home directory, or, perhaps better, especially for production servers, you can install it in /opt/relish .

Home directory installation:

   mkdir ~/relish
   cd ~/Downloads   # or wherever your binary tarball file was downloaded to.
   mv relish_<version>.<platform>.tar.gz ~/relish   # on MACOSX, the .gz has already been stripped
   cd ~/relish
   tar zxf relish_<version>.<platform>.tar.gz  # on MACOSX, xf relish_<version>.<platform>.tar
   sudo ./install.sh    # creates symbolic links in /usr/local/bin to ~/relish/bin/ executables 
                        # (puts relish command in your PATH)
   relish shared.relish.pl/hello_application

opt directory installation:

   sudo mkdir -p /opt/relish
   sudo chmod go+wx /opt/relish   
   cd ~/Downloads    # or wherever your binary tarball file was downloaded to.
   mv relish_<version>.<platform>.tar.gz /opt/relish   # on MACOSX, the .gz has already been stripped
   cd /opt/relish
   tar zxf relish_<version>.<platform>.tar.gz  # on MACOSX, xf relish_<version>.<platform>.tar
   sudo ./install.sh    # creates symbolic links in /usr/local/bin to /opt/relish/bin/ executables 
                        # (puts relish command in your PATH)
   relish shared.relish.pl/hello_application

Further installation instructions can be found in README_INSTALL_BINARY_DISTRIBUTION.txt in the root directory of the binary distribution.

Source Code Distribution

The relish software project is hosted at http://code.google.com/p/relish/

To compile the relish programming language, you will first have to install the Go programming language version 1.1 or 1.2 (http://golang.org), and if on linux, sqlite3 (3.5+) and libsqlite3-dev (sqlite3 C header files). 

	
   sudo mkdir -p /opt/devel
   sudo chmod go+wx /opt/devel
   cd /opt/devel
   hg clone https://code.google.com/p/relish/ 
   export GOPATH=/opt/devel/relish
   go get code.google.com/p/gosqlite/sqlite  
   cd relish
   go install relish/relish
   sudo ./install.sh    # creates symbolic links in /usr/local/bin to /opt/devel/relish/bin/ executables 
                        # (puts relish command in your PATH)
   relish shared.relish.pl/hello_application

Further installation and build instructions can be found in README_INSTALL_SOURCE_DISTRIBUTION.txt in the root directory of the source distribution.
