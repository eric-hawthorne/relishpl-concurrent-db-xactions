Source Code Distribution

The relish software project is hosted at http://code.google.com/p/relish/

To compile the relish programming language, you will first have to install the Go programming language (http://golang.org), the gosqlite package, sqlite3 (3.5+) and libsqlite3-dev (sqlite3 C header files).

Installation and build instructions can be found in README_INSTALL.txt in the root directory of the source distribution.

Binary Distributions

It should shortly be possible to download a relish binary distribution. Initially, these binary distributions will still require separate installation of sqlite3 (version 3.5 or greater) if sqlite3 is not already present on the platform, but will not require the installation of the Go programming language.

Binary distributions are planned in the relatively near term for linux i386, linux amd64, and macosx.

The relish compiler-interpreter is free software licensed with GPL version 3 and libraries for programmers are licensed with LGPL v3.
