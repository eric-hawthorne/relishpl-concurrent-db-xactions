relish Syntax and Semantics By Example - From the Inside Out
Shows and explains language constructs, starting from the most basic expressions and statements, proceeding on to control constructs and beyond.
Also contains a list of relish built-in methods available for use by programmers.
relish Roadmap
A list of relish programming language features planned but not yet implemented in the current version of the language. The list is aspirational. The order of listing in the roadmap is only an approximate indication of the order in which the features may come to fruition in the language.
Known Limitations
relish is definitely usable, but is at a formative stage of development. It has known limitations, as listed here. See also https://code.google.com/p/relish/issues/list
relish, being as it is at version 0.1.2 or so, is also subject to change.
No doubt relish also currently features Rumsfeldian unknown unknowns. If you find a problem or have a suggestion for improvement of the relish language, libraries, or programming environment/tools, please report it at the code.google.com link above.
You might still find relish, as is, worthwhile. It is being used in several active web-app projects, including a couple of web-apps in production.
