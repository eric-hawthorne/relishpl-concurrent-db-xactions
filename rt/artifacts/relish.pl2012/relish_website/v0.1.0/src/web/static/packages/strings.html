Summary

Methods to manipulate or find out about text strings. The String data type in relish is based on the string data type in Go. Strings are byte arrays, each containing a sequence of UTF-8 encoded unicode codepoints. Strings are immutable, so methods returning some modfification/function of a String are returning new copies or immutable slices of the original String. As well as the methods in the strings package, listed below, relish also provides built-in len, gt, lt, and eq methods for Strings.

base64Decode - decodes a base-64 encoded string; returns Bytes
base64Encode - encodes Bytes to produce a base-64 encoded String
base64Hash - the SHA-256 hash of the string as a base-64 encoded string
cat - concatenates string representations of argument values; returns resulting String
contains - whether a string contains a specified substring
fill - substitutes argument values into %s placeholders in the string, returns resulting string
first - the prefix of the string containing n utf8-encoded unicode codepoints
hasPrefix - true if the argument is a prefix of the string
hasSuffix - true if the argument is a suffix of the string
hexHash - the SHA-256 hash of s as a lowercase-hexadecimal-encoded string
index - the index of the first occurrence of the substring in the string
join - concatenates list of values to create a single string; optional separator between elements
last - the suffix of the string containing the last n utf8-encoded unicode codepoints
lastIndex - the index of the last occurrence of the substring in the string
lower - a copy of the string s with all Unicode letters mapped to their lower case
numCodePoints - the number of utf-8 encoded unicode codepoints (i.e. unicode characters)
replace - a copy of the string with instances of old replaced by new
slice - substring of the string, specified by an inclusive start and (optional) exclusive end index
split - splits the string into a list of substrings that were separated by the specified separator.
title - a copy of the string with first letters of words title-cased and other letters lowercased
trimSpace - a slice of the string s, with all leading and trailing white space removed
upper - a copy of the string s with all Unicode letters mapped to their upper case
urlPathPartEncode - a copy of the string s encoded to be part of a url path
urlPathPartDecode - a string that is the decoding of an urlPathPartEncode'd string

base64Decode s String urlSafe Bool > decoded Bytes err String
"""
 Decodes a base-64 encoded string; returns Bytes.
 If urlSafe is true, assumes the string was encoded with the URL-safe and filename-safe encoding 
 format which uses an alphabet in which codes 62 and 63 are -_
 If urlSafe is false, assumes the string was encoded with the original base-64 encoding format 
 which uses an alphabet in which codes 62 and 63 are +/ 
"""


base64Encode s String urlSafe Bool > decoded Bytes err String
base64Encode s String urlSafe Bool lineLength Int > decoded Bytes err String
"""
 Encodes Bytes to produce a base-64 encoded String.
 If urlSafe is true, uses the URL-safe and filename-safe encoding format which
 uses an alphabet in which codes 62 and 63 are -_
 If urlSafe is false, uses the original base-64 encoding format which
 uses an alphabet in which codes 62 and 63 are +/ 
 If lineLength is supplied, the output string is justified into lines of the specified number
 of characters.
"""


base64Hash s String > String
"""
 Returns the SHA-256 hash of s as a base-64 encoded string.
"""


cat ...obj Any > String
"""
 Concatenates the String representation of up to 9 argument objects together and returns the resulting String.
 Will eventually accept an arbitrary number of objects as arguments. 
"""


contains s String substr String > Bool 
"""
 Returns whether s contains the substring substr.
"""


fill s String ...obj Any > String
"""
 Substitutes objects (currently up to 9) into placeholders in the string s, returning the resulting string.
 Each placeholder is denoted by a %s inside the string s. The number of object arguments other than s
 and the number of placeholders in s must be the same.
 The default String representation of each object is used.
"""

Examples:

   firstName = "John"
   lastName = "Doe"
   s = fill "Hello, my name is %s %s and I am %s years old." firstName lastName 19
   print s

   // Will print:    
Hello, my name is John Doe and I am 19 years old.

   s = fill """
"""
Hello!

My name is %s %s 
and I am %d years old.
"""
            firstName
            lastName
            19
   print s
	
   // Will print:
Hello!

My name is John Doe 
and I am 19 years old.	


first s String n Int > String
"""
 Returns the substring of the string s beginning at beginning of s and containing n utf8-encoded unicode 
 codepoints.
 If s has fewer codepoints in it than n, returns the whole string s and does not complain.
 Note: slice indexes are counting bytes of the string. This method is counting utf8-encoded unicode 
 characters.
"""


hasPrefix s String substr String > Bool
"""
 Returns true if substr is a prefix of s (or all of s). false otherwise.
"""


hasSuffix s String substr String > Bool
"""
 Returns true if substr is a suffix of s (or all of s). false otherwise.
"""


hexHash s String > String
"""
 Returns the SHA-256 hash of s as a lowercase-hexadecimal-encoded string. 
 Less compact than base64 but contains only alphanumeric characters.
"""


index s String substr String > Int
"""
 Returns the index (counting in bytes from 0) of the first occurrence of the substring in the string s, or -1 if substr is 
 not found in s.
"""


join a []String > String
"""
 Concatenates the elements of a to create a single string. 
"""


join a []String sep String > String
"""
 Concatenates the elements of a to create a single string. The separator string sep is placed between 
 elements in the resulting string.
"""


join a []Any sep String > String
"""
 Concatenates the elements of a to create a single string. The separator string sep is placed between 
 elements in the resulting string.
 Note that list arguments are not covariant by element type yet, so you either have to supply
 a []String or a []Any  - nothing else in between will work yet.
"""


last s String n Int > String
"""
 Returns the suffix of the string s containing the last n utf8-encoded unicode codepoints in s.
 If s has fewer codepoints in it than n, returns the whole string s and does not complain.
 Note: slice indexes are counting bytes of the string. This method is counting utf8-encoded unicode 
 characters.
"""


lastIndex s String substr String > Int
"""
 Returns the index (counting in bytes from 0) of the last occurrence of the substring in the string s, 
 or -1 if substr is not found in s.
"""


lower s String > String
"""
 Returns a copy of the string s with all Unicode letters mapped to their lower case.
"""


numCodePoints s String > Int
"""
 Returns the number of utf-8 encoded unicode codepoints (i.e. unicode characters) in the String. 
 This is not the byte-length of the string, which may be greater than the number of encoded codepoints.
"""


replace s String old String new String > String
"""
  Returns a copy of the string s with all non-overlapping instances of old replaced by new. 
"""


replace s String old String new String n Int > String
"""
  Returns a copy of the string s with the first n non-overlapping instances of old replaced by new. 
  If n < 0, there is no limit on the number of replacements.
"""


slice s String start Int end Int > String
"""
 Returns the substring of s beginning at index 'start' (counting in bytes) and ending just before 'end'.
 If 'end' is negative, the substring ends at the position such that the -'end' last bytes of
 the original string are not included in the substring. So to remove ".doc" use 
 slice fileName 0 -4
"""


slice s String start Int > String
"""
 Returns the substring of s beginning at index 'start' (counting in bytes) and including all
 subsequent bytes of the string.
"""


slice s String start Int end Int > String
"""
 Returns the substring of s beginning at index 'start' (counting in bytes) and ending just before 'end'.
 If 'end' is negative, the substring ends at the position such that the -'end' last bytes of
 the original string are not included in the substring. So to remove ".doc" use 
 slice fileName 0 -4
"""


split s String sep String > [] String
"""
 Returns the list of substrings of s which were separated in s by the specified separator.
 If the separator occurs at the beginning of s, an empty string is the first list member.
 If the separator occurs at the end of s, an empty string is the last list member.
 If the separator does not occur in s, returns a list containing s.
 If the separator is the empty string "", s is split into each UTF-8 sequence 
 (i.e. each unicode character)
"""


title s String > String
"""
 Returns a copy of the string s with all Unicode letters that begin words mapped to their title case,
 and all other Unicode letters mapped to their lower case.
"""


trimSpace s String > String
"""
 Returns a slice of the string s, with all leading and trailing white space removed, as defined by Unicode.
"""


upper s String > String
"""
 Returns a copy of the string s with all Unicode letters mapped to their upper case.
"""


urlPathPartEncode
   s String 
   encodeDash Bool = true 
> 
   String
"""
 Returns a copy of the string s with the following characters mapped to %XX where XX is the hexadecimal value
 of the character's code point.

 Characters that will be escaped (encoded):

  '$','&','+',',','/',':',';','=','?','@','"','\'','<','>','#','%','{','}','|','\\','^','[',']','`'

  codepoint < 32 (decimal), codepoint > 127 (decimal)

 Only works for characters whose code point is in the range 0..255 (decimal) that is 0..FF (hexadecimal).
 Specifically does not work for unicode characters with code point > 255. These characters are always invalid 
 in the path part of a URL.

 If encodeDash is true (the default), then spaces ' ' in the input string will be encoded as '-', whereas
 an actual '-' will be encoded as "*~*" and '.' will be encoded as "_~*"
This makes nice looking url path parts if the problem would just have
 been spaces. If encodeDash is false, spaces ' ' are encoded as %20 and '-' and '.' are not encoded.
"""


urlPathPartDecode
   s String 
   decodeDash Bool = true 
> 
   String
"""
 Returns a copy of the string s with each %XX escape sequence (where XX is the hexadecimal value
 of a character's code point) converted back into the original character.

 If decodeDash is true (the default), then '-' in the string will be decoded to spaces ' '
 and '*~*' in the string decoded to '-'
 and '_~*' in the string decoded to '.'
 If decodeDash is false, '-' will be left as is and no other decodings will be done.

 Note: If a URL path has been decomposed by the relish web request handling infrastructure into 
 individual argument values (or a list of argument values), those argument values will already
 have been PARTIALLY decoded, in that the %XX escape sequences will have been decoded, but '-'
 will have been left intact. So further processing will be required. E.g. to get a fully decoded
 string value, one would do the following to one of these argument values:

 s = urlPathPartDecode urlPathPartEncode argValue false
  
"""
