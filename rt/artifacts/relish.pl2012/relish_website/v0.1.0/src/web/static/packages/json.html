Summary

Methods to marshal data into a JSON-formatted serialized form, and to recreate relish objects from JSON-formatted data. Pattern of use would be e.g. 


import
   json


serializeAndBack students [] Student > [] Student   
"""
 Sends some students through the transporter beam, and recreates them
 as new objects somewhere else.
"""
   jsonText err = jsonMarshal students 

   reconstitutedStudents = []Student
   reconstitutedStudents err = jsonUnmarshal jsonText reconstitedStudents
   => reconstitutedStudents

jsonMarshal - convert an object (tree) to JSON text.        
jsonUnmarshal - convert JSON text into an object (tree).        

Methods


jsonMarshal val Any > jsonEncoded String err String
"""
 Marshals the argument value/object/collection including only the public
 attributes of structured objects. 
 The object tree cannot contain reference loops or multiple references to the same object.
"""


jsonMarshal val Any includePrivate bool > jsonEncoded String err String  
"""
 Marshals the argument value/object/collection including the public
 and if indicated also the private attributes of structured objects.
 The object tree cannot contain reference loops or multiple references to the same object.
"""


jsonUnmarshal json String > val Any err String 
"""
 Decodes the json-encoded string argument into a relish object or object tree, which is returned.
"""


jsonUnmarshal json String obj Any > obj Any err String 
"""
 Decodes the json-encoded string argument into a relish object or object tree, which is returned.
 Attempts to populate the supplied object (second argument), 
 which must be a collection, a map, or a structured object. The object argument itself
 is returned, after being populated with attribute/collection values from the JSON string.
"""
